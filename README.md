# vue-vite-genesis
super minimal vue project starter based off [vue3-realworld-example-app](https://github.com/mutoe/vue3-realworld-example-app) 

## setup

```zsh
npm run install

# local dev
npm run dev

# dist build
npm run build

# lint
npm run lint

# tests
npm run test

```
